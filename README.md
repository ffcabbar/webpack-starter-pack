# webpack-starter-pack
:fire: A simple webpack starter kit for react.


### Installation

```
npm install
```

### Start Dev Server

```
npm run start
```

### Build Prod Version

```
npm run build
```

### Requirements
* [Node](https://nodejs.org) > 10.0

### Features:

* ES6 Support via [babel](https://babeljs.io/) (v7)
* SASS Support via [sass-loader](https://github.com/jtangelder/sass-loader)
